**华为（外部优先）**

> Direct为直连链路，其中除了直连链路外其余优先级均可用户手工修改，优先级越小越优；

| 协议类型  | 优先级 |
| --------- | ------ |
| Direct    | 0      |
| OSPF      | 10     |
| IS-IS     | 15     |
| Static    | 60     |
| RIP       | 100    |
| OSPF ASE  | 150    |
| OSPF NSSA | 150    |
| IBGP      | 255    |
| EBGP      | 255    |



华为（内部优先级）

> 选择路由时会优先比较外部优先级，而当不同的路由协议人为操作使其相同优先级后，系统会根据内部优先级决定哪个路由协议的路由被优选。而若内外优先级均相同，系统选择链路Cost较小一方，而Cost若相同，则形成Balance路由；

| 协议类型      | 优先级 |
| ------------- | ------ |
| Direct        | 0      |
| OSPF          | 10     |
| IS-IS Level-1 | 15     |
| IS-IS Level-2 | 18     |
| Static        | 60     |
| RIP           | 100    |
| OSPF ASE      | 150    |
| OSPF NSSA     | 150    |
| IBGP          | 200    |
| EBGP          | 20     |
